Quantum Neural Network (QNN)¶

The quantum neural network (QNN) tries a different approach. It needs already precalculated boundaries and then tries to further improve the result.

quantimize.quantum.quantum_neural_network.quantum_neural_network(flight_nr, n_qubits, init_solution)¶

Create a quantum neural network for a given flight with n_qubits and an initial solution.

Parameters

  • flight_nr (int) – flight number

  • n_qubits (int) – number of qubits the QNN should operate on

  • init_solution (list) – list of initial boundary points

Returns

result of the optimization containing the boundaries and the cost

Return type

scipy.optimize
