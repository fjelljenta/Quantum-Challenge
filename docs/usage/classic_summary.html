Access the classic functions¶

The access to classic functions is there to simplify calls from a higher level like the Quantimize.ipynb. Some are just wrappers, but other add additional functionality to check for errors or wrap results in a dictionary.

quantimize.classic_summary.compute_cost(trajectory)¶

Wrapper for the computation of the cost function

Parameters

trajectory (list) – List of trajectory points

Returns

Cost of the flight

Return type

float

quantimize.classic_summary.genetic_algorithm_solution(flight_nr, dt=15, **kwargs)¶

Returns the genetic algorithm solution for a given flight and maps it to constant time points

Parameters

flight_nr (int) – Flight number

Returns

Trajectory in the format of a list of tuples (long, lat, time) embedded in a dict with the flight number

Return type

dict

quantimize.classic_summary.straight_line_solution(flight_nr, dt=15, **kwargs)¶

Returns the straight line solution for a given flight

Parameters

  • flight_nr – Flight number

  • dt – The time step in seconds

Returns

Trajectory in the format of a list of tuples (time, longitude, latitude) embedded in a dict with the flight number
