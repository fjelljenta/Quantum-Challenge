Creation of JSON data¶

This section is dedicated to creating JSON files out of the data given, so the access in Python can be simplified. Therefore our code offers different conversion functions, namely:

quantimize.create_json_data.convert_hPa_to_FL(hPA)¶

Converts hPa to flight level according to the table given in the PDF

Parameters

hPA (int) – Flight level in hPA

Returns

Flight level in FL

Return type

int

quantimize.create_json_data.create_atmo_data()¶

Converts the atmospheric data to json file

quantimize.create_json_data.create_bada_data()¶

Creates the flight level information as json file

quantimize.create_json_data.create_flight_schedule()¶

Creates the flight schedule as json file

For the functions to work, there need to be a folder called data and the file names are also hard coded, which means, they need to match the original description.
